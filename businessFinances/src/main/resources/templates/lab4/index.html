Lab 4 - Процедуры

  • Transaction Summary

    Обзор всех транзакций, связанных с указанным бизнесом, с ключевыми деталями и суммами.

  • Top Agents by Transaction Volume

    Рейтинг агентов, участвующих в транзакциях, отсортированных по объему работы.

  • Monthly Transactions Report

    Ежемесячный отчет о транзакциях, показывающий активность бизнеса по месяцам.
